# 学习区说明

这个目录只放你的学习成果，不改动原仓库的指南内容。

## 这个仓库原本怎么组织

`guide_zh.md` 目前大致分成这几层：

1. 考试简介、候选人画像、考试形式、考试场景
2. 官方文档入口
3. 理论基础
4. 按考试领域整理的考点注释
5. 题目、模拟测试、实践练习
6. 附录与备考建议

它适合“通读”和“查资料”，但不太适合你自己长期复习，因为原文的信息密度太大，理论、考点、题目都混在一份长文里。

## 我们的笔记结构建议

学习笔记建议分成 4 层：

1. 路线图
2. 主题笔记
3. 题目与错题
4. 复盘与速记

对应目录如下：

```text
study/
  README.md
  00-学习路线.md
  01-主题笔记/
  02-题目与错题/
  03-复盘速记/
```

## 每篇主题笔记怎么写

每篇笔记固定按这个模板写，避免越记越乱：

1. 这是什么
2. 它解决什么问题
3. 核心概念/核心参数
4. 一个最小示例
5. 容易错的点
6. 考试会怎么考
7. 一句话总结

这样记的好处是：

- 先建立直觉，不会一上来陷进细节
- 每篇都能直接复习，不依赖回原文翻半天
- 后面做题时，能把“概念”和“陷阱”快速对上
